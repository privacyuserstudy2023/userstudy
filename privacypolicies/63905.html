Privacy Policy

All data collected by the Alexa Skill "Phone Finder" and the Android App "Alexa Phone Finder" are only used for running and improving the software. No data will be shared with any third parties.
Only anonymous data is collected. It consists of the Amazon Alexa "User ID" and the "Android App Token". Both can not be related to any personal data. They are solely used to connect the Alexa Skill with the Android App.
All data processing and storage is done on Amazon AWS servers. Those reside in the European Union.
The following data is processed:

  • Requests to the skill
  • Date of usage
  • Error and Debug Messages

All data is transmitted over SSL/TLS. This privacy policy can change, e.g. when the functionality of the software changes. The current version can be found here.
